# Скелет презентации — 12 слайдов

Стратегия колоды — **«Бизнес в коробке» (вертикальный конструктор)**, `strategy_vertical_constructor.md`. Каждый слайд — файл `NN_slug.md` с одной мыслью и содержанием.

Сквозной нарратив: **вызов → кто клиент и что его мучает → куда фокус → рынок (красный океан) и вывод → концепция «бизнес в коробке» → путь клиента (3 подсегмента) → что внутри коробки (ядро + вертикали) → AI-операционный директор (запускает и не даёт упасть) → продукты → удержание/продвижение/эффект → пилот и закрытие.**

## Как читать таблицу

- «Кейс» — номер пункта постановки задачи из `championship/case.md`.
- «Критерии» — номер критерия из `championship/criteria.md` (Кр.1 ЦА, Кр.2 рынок, Кр.3 стратегия, Кр.4 структура, Кр.5 презентация, Кр.6 ИИ). Кр.4 и Кр.5 — сквозные требования ко всей колоде, проверяются по `championship/checklist.md` (разделы 11–12), не привязаны к одному слайду.
