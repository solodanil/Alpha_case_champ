# Presentation

Итоговое решение в виде тезисов.

- `full_deck.md` — сборный markdown всей презентации (единый нарратив)
- `slides/` — по одному markdown-файлу на каждый слайд (`01_title.md`, `02_challenge.md`, ... `12_pilot_closing.md`; скелет и привязка к критериям — `00_outline.md`) — удобно для точечной доработки, оценки и итераций по конкретному слайду

Презентация как визуальный артефакт (слайды в PowerPoint/Figma/etc.) верстается отдельно на основе этих markdown-файлов — не генерируется одним промптом целиком.
